JInterface2 by zzj1213

Enhanced JInterface, with conversion to standard Java type like ArrayList, HashMap and JavaBean. It supports nested List and Tuple. It can be used as a dropin replacement for existing OtpErlang.jar.

The following are the Erlang and Java type conversion cases: (TBD).

Dependencies

1. commons-beanutilits-1.7.0.jar 2. commons-logging-1.1.jar 3. commons-pool-1.3.jar 4. javolution-5.4.3.jar

Install

Dropin replacement for original OtpErlang.jar. Fully compatible with the original JInterface lib.
